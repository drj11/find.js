# find.js

POSIX compliant find in node.js (not POSIX
compliant yet)

# Where

https://github.com/drj11/find.js

# Implementation Status

## Implemented

* `-a` and `-o` with correct precedence
* `-type f` and `-type d` and `-type l` and `-type p`
* `-exec` when terminated with ';'
* `-links`
* `-print`
* `-prune`
* `-size`
* `-depth`
* implicit `-print`
* implicit `-a`
